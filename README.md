# Kaptcha project 


### 1. Клонирование репозитория
```bash
git clone 
```

### 2. Создание виртуального окружения
```bash
python -m venv venv
source venv/bin/activate  # Для macOS/Linux
venv\Scripts\activate # Для Windows
```

### 3. Установка зависимостей
```bash
pip freeze > requirements.txt
```
